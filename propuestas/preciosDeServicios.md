# Precios y Argumentos — GatorojoLab (Versión Detallada)

Este documento describe de forma completa los servicios, rangos de precio, tiempos, entregables, requisitos del cliente y limitaciones.

---

# 0. Desarrollo de Aplicaciones Web JAMStack

**Rango:**

- Proyecto estándar: **US$1,000 – US$7,000**
- Por hora: **US$40 – US$80**

## Qué incluye

- Arquitectura JAMStack (React, Astro, Waku, TypeScript).
- Integración con APIs externas (REST, Firestore, Firebase).
- Optimización SEO técnico (sitemap, meta tags, accesibilidad básica).
- Diseño UI ligero o uso de un sistema de componentes ya definido.
- Configuración de hosting moderno (Vercel, Netlify).
- Formularios(1 a 5, según tamaño del proyecto), autenticación básica(correo y contraseña), dashboards simples.
- Versionamiento y repositorio Git público o privado.
- **Aplicación simple: 3-5 páginas/vistas**
- **Aplicación mediana: 6-10 páginas/vistas**
- **Proyecto avanzado: 10+ páginas/vistas**

## Tiempo estimado

- Aplicación simple: **2–3 semanas**
- Aplicación mediana: **5–10 semanas**
- Proyecto avanzado: **10–20+ semanas**

## Entregables

- Código fuente completo.
- Documentación básica de instalación y despliegue.
- Acceso al panel de hosting.
- Manual corto de uso (interno o para cliente).
- Componentes reutilizables documentados.

## Limitaciones típicas

- No incluye backend complejo hecho a medida (solo integraciones).
- No incluye diseño de marca o identidad visual.
- No incluye infraestructura avanzada (servidores dedicados).
- Límite razonable de revisiones (2 o 3 rondas según tamaño del proyecto).

## Qué debe proporcionar el cliente

- Contenido (textos, imágenes, videos).
- Documentación de APIs externas.
- Credenciales de hosting o servicios de terceros.
- Aprobaciones en tiempos razonables para evitar retrasos.

## Extras cobrables aparte

- Panel de administración personalizado.
- Autenticación avanzada (roles, permisos, SSO).
- Integraciones con servicios premium (Stripe, HubSpot, Airtable).

# 1. Integración de Componentes con IA

**Rango:**

- Chatbot básico: **US$100 – US$500** (alimentado estáticamente, faq corto o documentos pequeños)
- Chatbot avanzado: **US$500 – US$2,000** (alimentado dinámicamente con Firestore)
- Consultoría: **US$50 – US$150/h**

## Qué incluye

- Implementación de API de OpenAI u otros modelos.
- Chatbots contextuales.
- Asistentes para clientes (FAQs, soporte, guía de navegación).
- Generadores de contenido (texto, resúmenes, emails).
- Integraciones con Firestore.
- NO incluye costo mensual de API:
  - En caso de requerir uso de OpenAI u otro proveedor, se aplican los siguientes costos aproximados:
  - $5: 8400 interacciones individuales aprox.
  - $10: 17000 interacciones individuales aprox.
  - $20: 34000 interacciones individuales aprox.
  - Si hay exceso en uso, se cobrará aparte según tarifa de OpenAI u otro proveedor.

## Tiempo estimado

- Integración simple: **3–7 días**
- Asistente avanzado: **2–6 semanas**

## Entregables

- Código + integración funcional.
- Documentación para el cliente.
- Panel de configuración (para chatbot avanzado).
- Ajuste de prompts o archivo estático de conocimiento mensual (1 vez al mes durante el primer año de uso).

## Limitaciones

- No incluye entrenamiento de modelos grandes (solo prompts + contexto).
- Los costos de la API son responsabilidad del cliente.
- No incluye infraestructura de seguridad empresarial sin cotizar aparte.

## Qué debe proporcionar el cliente

- Claves API.
- Base de conocimiento si es necesaria (PDFs, FAQ, documentación).
- Acceso a base de datos o endpoints.
- Acceso al repositorio y hosting si es necesario.

## Extras

- Dashboard para monitoreo del uso.
- Fine-tuning si la plataforma lo permite.

# 2. Análisis y Optimización de Rendimiento Web

**Rango:**

- Auditoría: **US$250 – US$900**
- Optimización completa: **US$1,000 – US$3,500**

## Qué incluye

- Auditorías con herramientas pro.
- Mejora de Core Web Vitals.
- Optimización de imágenes, scripts y fuente de datos.
- Refactorización ligera.
- Reporte antes/después con métricas.

## Tiempo estimado

- Auditoría: **5–10 días**
- Optimización completa: **2–8 semanas**

## Entregables

- Reporte completo (PDF o markdown).
- Lista de mejoras aplicadas.
- Recomendaciones adicionales.
- Capturas de métricas antes/después.

## Limitaciones

- No incluye rediseño total de UI.
- No incluye reescritura completa del sistema.
- Sitios extremadamente lentos pueden requerir un proyecto mayor.

## Qué debe proporcionar el cliente

- Acceso al código o repositorio.
- Acceso al hosting o panel.
- Prioridades del negocio y páginas clave.

## Extras

- Implementación CDN profesional.
- Conversión a JAMStack.
- Reducción de costos de infraestructura.

# 3. Auditoría UI/UX

**Rango:**

- Sitio pequeño: **US$300 – US$1,200**
- Producto grande: **US$1,200 – US$4,000**

## Qué incluye

- Revisión de usabilidad y accesibilidad.
- Evaluación heurística profesional.
- Revisión de flujo de usuario y conversiones.
- Recomendaciones priorizadas.
- Evidencia en video o capturas.

## Tiempo estimado

- Sitio pequeño: **1-2 semanas**
- Producto grande: **3–6 semanas**

## Entregables

- Informe detallado (PDF o markdown).
- Checklist de accesibilidad WCAG AA.
- Ejemplos y propuestas para mejorar UI.
- Prioridad de cambios (alto/medio/bajo).

## Limitaciones

- No incluye rediseño visual.
- No incluye implementación del código.
- Revisiones limitadas (1 ronda).

## Qué debe proporcionar el cliente

- Acceso al producto o prototipo.
- Objetivos del negocio.
- Métricas actuales si existen (Google Analytics, Hotjar).

## Extras

- Rediseño parcial o completo basado en la auditoría.
- Seguimiento mensual.
- Pruebas de usabilidad reales.

# 4. Desarrollo y Hospedaje de Sitios Web

## Desarrollo

**Rango:**

- Sitio simple: **US$400 – US$1,600**
- Sitio corporativo: **US$1,600 – US$4,500**
- Sitio avanzado: **US$3,000 – US$7,000**

### Qué incluye

- Diseño + desarrollo JAMStack.
- Estructura web completa: Home, Servicios, Contacto, Blog.
- Optimización SEO.
- Integración de formularios y analíticas.
- Hosting configurado.
- Compra de dominio (si aplica y costo a cargo del cliente).
- Soporte para lanzamiento inicial y monitoreo durante la primera semana.

## Definición de tamaños

- **Sitio simple: 3-5 páginas, conexión a máximo 2 APIs**
- **Sitio corporativo: 6-12 páginas, conexión a máximo 4 APIs**
- **Sitio avanzado: 12+ páginas, conexión a máximo 10 APIs**

### Tiempo estimado

- Sitio simple: **2–4 semanas**
- Corporativo: **4–8 semanas**
- Avanzado: **12–24 semanas**

### Entregables

- Sitio funcionando.
- Código fuente.
- Manual básico.
- Dashboard analítico (no aplica para sitios simple).
- Análisis básico de tráfico y comportamiento de usuarios durante 1 mes después del lanzamiento.

### Limitaciones

- No incluye branding.
- No incluye e-commerce complejo.
- No incluye contenido (solo estructura).
- Revisión limitada (2 rondas).
- El costo cubre hasta 6 meses de hosting básico. Después de este período, se aplican cargos **mensuales** según el tamaño del proyecto:
- Básico: US$10 – US$35
- Corporativo: US$50 – US$200
- Avanzado: US$200 – US$600

### Cliente debe proporcionar

- Textos, fotos, identidad visual.
- Dominio o acceso al dominio.
- Aprobaciones rápidas.

### Extras

- Blog con CMS.
- Secciones adicionales.
- Multilenguaje.
- Dashboard analítico avanzado.
- Correos corporativos.
- Análisis avanzado de tráfico y comportamiento de usuarios.
